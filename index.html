Cours GIT

Qu'est-ce que GIT

GIT est un outils de versionning et il permet d'avoir une tracabilité et un historique tous les fichiers du projet

Il permet également de travailler en équipe: Envoyer nos modifications a nos collègue et récupérer les modifications des collegues.

Dans la tracabilité des fichiers GIT nous donne toujours 3 infos : Qui, Quand, Pourquoi ?

Installer et configurer GIT

URL de téléchargement : https://git-scm.com
git config --global user.email 'bercy.cedric@hotmail.fr'
git config --global user.name 'CédricBercy'

Les Commandes de base

Initialiser un projet:
git init
Voir le statut des fichiers
git status
Ajouter des fichiers
git add 'fichier'
Ajouter TOUS LES FICHIERS :
git add .
Commit (enregistrer) les fichier ajouter
git commit
Un commit nécéssite toujours un message
commit avec message:
git commit -m 'nom du fichier'
commit avec add + message (fonctionne sur les fichiers déja add précédemment):
git commit -a -m 'nom du fichier'

Correction des fichiers

Voir les différents commit :
git log
Corriger les modifications sur un fichier (pas encore add) :
git checkout 'fichier'
